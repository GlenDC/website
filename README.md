# Plabayo Website

The source code of the Plabayo website.

https://www.plabayo.tech

## Build Locally

Install Zola first:

> <https://www.getzola.org/documentation/getting-started/installation/>

Build:

```
$ zola build
```

Or serve:

```
$ zola serve --interface 127.0.0.1 --port 8080
```
